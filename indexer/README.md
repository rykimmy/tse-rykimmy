### Compiling

If executing 'indexer' caller should:

1. `make clean; make` in the tse-xxxx directory
2. `cd indexer`
3. run indexer (see indexer.c for usage)

If running 'testing.sh' which runs test cases of indexer, caller should:

1. `make clean; make` in the tse-xxxx directory
2. `cd indexer`
3. `make test`
